# Contributing

Hydra's `main` branch is the release branch. Until GitHub branch protection is available for this private repository, treat these rules as mandatory project policy:

1. Work on a feature branch.
2. Open a pull request into `main`.
3. Wait for the `build` GitHub Actions check to pass.
4. Request owner review before merging.
5. Do not force-push or delete `main`.

Direct pushes to `main` should be reserved for emergency owner fixes only, and the follow-up pull request or commit notes should explain why the normal PR path was bypassed.
