# Support

Hydra is a public beta. Please file issues at:

https://github.com/Geraldlol/hydra/issues

Include:

- VS Code version and operating system.
- Hydra version.
- Whether the failure happened in one-shot or terminal-bridge transport.
- The output of `Hydra: Run Doctor`.
- Any relevant `.hydra/support-bundle.md` excerpt with secrets removed.

Do not paste Codex, Claude, Telegram, GitHub, or Marketplace tokens into issues.
